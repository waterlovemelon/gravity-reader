# 预设背景图片说明

本目录存放阅读器的预设背景图片，共需 5 张 JPG 格式图片：

## 需要的文件

1. `paper_texture.jpg` - 纸质纹理
2. `book_page.jpg` - 书页效果
3. `warm_beach.jpg` - 温暖沙滩
4. `classic_parchment.jpg` - 经典羊皮纸
5. `subtle_grain.jpg` - 细腻颗粒

## 图片规格

- 格式：JPG
- 分辨率：建议 1920x1080 或更高
- 质量：85-95%
- 文件大小：单个文件建议不超过 2MB

## 说明

这些图片会作为用户可选的预设背景显示在「亮度与颜色」面板中。
用户也可以自己上传自定义图片（最多 10 张）。

请在此目录中放置对应的图片文件，或者在 `reader_background.dart` 中修改预设图片路径。
